# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| `main`  | ✅ Yes (updates)   |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue. Instead, send an email to **security@autoflow.dev** with a clear description of the issue.  We will acknowledge receipt within 48 hours and aim to issue a fix within 14 days.

## Responsible Disclosure

We appreciate responsible disclosure and will give proper credit to any researcher who helps improve AutoFlow's security.
